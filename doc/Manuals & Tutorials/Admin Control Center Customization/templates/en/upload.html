The following file types can be applied:
  • A V2C file contains a license update from your software vendor, or a firmware update for your HASP HL keys.
    • {if_commuting}
  • An H2R file contains a detached license.
  • An R2H file contains a cancelled detached license (to be re-attached to its original key).
  • An ID file contains the identifiers of the Sentinel HASP Run-time on a remote machine (occasionally required for creating a detached license).
    • {endif}
  • An ALP file contains an Admin Control Center language pack.
